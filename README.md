# Landing Page - Punto Cero Collection
## Descripción 
“Punto Cero Collection” es una landing page enfocada en una marca de ropa de gimnasio que combina estilo, rendimiento y actitud. El diseño busca transmitir energía y motivación, destacando prendas ideales para entrenar con confianza y comodidad.
## Objetivo
Desarrollar una página que genere interés en el usuario, comunique la identidad de la marca y motive la interacción o posible compra de los productos
## Tecnlogias 
- VS
- HTML
- JAVASCRIPT
- CSS
## Uso 
Para visualizar la landing page, descarga los archivos del proyecto y abre el archivo index.html en un navegador web.
